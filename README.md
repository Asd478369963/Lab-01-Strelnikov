### Описание задания
Запросить строку и посчитать количество гласных букв русского алфавита (а, е, ё, и, о, у, ы, э, ю, я — без учёта регистра).

### Структура проекта
- `src/` — исходный код
- `src/ConsoleApp/` — консольное приложение с решением задачи
- `Lab-01-Strelnikov.sln` — файл решения Visual Studio

### Порядок запуска
1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/Asd478369963/Lab-01-Strelnikov.git
   ```
2. Открыть `Lab-01-Strelnikov.sln` в Visual Studio или выполнить:
   ```bash
   dotnet run --project src/ConsoleApp/ConsoleApp.csproj
   ```
