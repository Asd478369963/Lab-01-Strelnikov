# Перечень артефактов проекта Lab-01-Strelnikov

В данном документе перечислены все ключевые материалы (артефакты), созданные и используемые в процессе разработки проекта.

## 1. Исходный код
- `src/ConsoleApp/Program.cs` — основной файл программы, координирующий работу модулей.
- `src/ConsoleApp/Modules/InputModule.cs` — модуль для взаимодействия с пользователем и ввода данных.
- `src/ConsoleApp/Modules/ValidationModule.cs` — модуль для проверки корректности введенных данных.
- `src/ConsoleApp/Modules/ProcessingModule.cs` — модуль для основной логики обработки данных (подсчет гласных) и вывода результата.

## 2. Документация и конфигурация
- `README.md` — общее описание проекта, инструкции по запуску и сведения об исполнителе.
- `.gitignore` — файл конфигурации Git для исключения временных файлов и результатов компиляции.
- `Lab-01-Strelnikov.sln` — файл решения Visual Studio для управления проектом.
- `src/ConsoleApp/ConsoleApp.csproj` — файл проекта .NET.

## 3. Визуализация архитектуры
- `lab2_diagram.png` — UML-диаграмма модулей, отображающая структуру и зависимости между компонентами проекта.

## 4. Протоколы и отчетность
- `ProjectArtifacts.md` — настоящий перечень всех артефактов проекта.
- `ProjectProtocols.md` — протокол действий и изменений, фиксирующий ход разработки.
- `Лабораторная_работа_1_Руководство.md` — пошаговое руководство по выполнению ЛР №1.
- `Лабораторная_работа_2_Руководство.md` — пошаговое руководство по выполнению ЛР №2.
